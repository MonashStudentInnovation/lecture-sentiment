# lecture-sentiment

The Code for analysing the sentiment during the talk

To install dependencies run

```
npm install

# or if you have yarn
yarn
```

To run project

```
npm start

# or

yarn start
```

# Developing

To develop on this visit `/src/config.js` and add both your Firebase API keys and configuration and your Google NLP API
